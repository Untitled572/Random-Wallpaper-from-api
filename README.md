# Random Wallpaper from API

一个轻量级的 Linux 桌面自动壁纸获取和切换程序，基于 Go 语言编写。

## ✨ 核心特性

* **极低占用**：Go 语言编译为单一二进制文件，告别 Python/Node.js 的臃肿内存占用。
* **智能尺寸过滤**：自动利用高速内存读取图片头信息，严格过滤并丢弃分辨率小于 `1920x1080` 的图片。
* **自动重定向处理**：原生支持处理 302 跳转类型的图片 API。
* **无感定时切换**：默认每 30 分钟自动在后台轮换一次壁纸。
* **自动空间清理**：自动清理历史文件，默认仅保留最近 3 张壁纸，防止占用硬盘。
* **系统托盘支持**：提供顶栏图标，支持“立即切换”和“打开图库”。
* **支持自定义API**： 默认api来自[随机二次元图片API文档](https://gitea.890100.xyz/yeqing/acg-api), 顶栏图标内可自定义
## 📦 安装与运行

需要安装以下依赖

```bash
sudo pacman -S --needed go base-devel gtk3 libappindicator-gtk3 zenity
```



本项目提供了一键构建与安装脚本。脚本会自动安装依赖、配置国内 Go 镜像源、编译代码并设置开机自启。

```bash
# 1. 赋予执行权限
chmod +x install.sh

# 2. 运行一键安装脚本
./install.sh

# 卸载
./uninstall.sh
```

本项目包含AI辅助创作
